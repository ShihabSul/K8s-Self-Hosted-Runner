# Run in a container

Pre-built containers with PandABlocks-rootfs-docs and its dependencies already
installed are available on [Github Container Registry](https://ghcr.io/shihab-dls/PandABlocks-rootfs-docs).

## Starting the container

To pull the container from github container registry and run:

```
$ docker run ghcr.io/shihab-dls/PandABlocks-rootfs-docs:latest --version
```

To get a released version, use a numbered release instead of `latest`.
